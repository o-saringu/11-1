プログラム復習コンテンツ

配列-1

サンプルコードの結果

サンプルコード

　　 float[] x=new float[3000];

　　 　

　　 void setup(){

　　 　size(240, 120);　

　　 　noStroke();

　　 　fill(255, 200);

　　 　for(int i=0; i　　 　　x[i]=random(-1000, 200);

　　 　}

　　 }

　　　

　　 void draw(){

　　 　background(0);　

　　 　for(int i=0; i　　 　　x[i]+=0.5;

　　 　　float y=i*0.4;

　　 　　arc(x[i], y, 12, 12, 0.52, 5.76);

　　 　}

　　 }

説明文

　配列は、沢山の変数を管理することができる。

　配列の詳しい説明はコチラ。

　

　今回のプログラムでは、3000個の図形が左から右に移動するプログラムである。

　3000個の変数を全て書くのは無謀であるため、配列を使用し管理している。

　補足：まずsetup()で3000個の図形をランダム関数により様々な場所に生み出している。

　　　　draw()では生みだした図形を左から右へ移動する処理を行っている。
